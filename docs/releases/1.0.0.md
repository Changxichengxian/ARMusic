# ARMusic 1.0.0

ARMusic 1.0.0 是基于 LMusic 1.5.4 改造后的第一个发布版本。

## 这版有什么

- Android 包名改为 `com.lalilu.lmusic.armusic`，不会覆盖原来的 LMusic。
- 内置音乐标签编辑，支持歌曲信息、歌词、封面的联网搜索和保存。
- 把原来的专辑整理方向改成作品整理，更适合动漫音乐库。
- 支持同曲分组，把同一首歌不同版本的播放次数和播放时长合并统计。
- 增加播放时长统计、播放次数排序显示和历史数据导入累加。
- 增加局域网同步入口，支持 Android 和桌面端互传新增歌曲。
- 桌面端改为 Tauri，提供本地扫描、基础播放和同步服务。
- 增加准备听、动漫、漫画、小说四类备忘。
- 增加屏蔽文件夹能力，减少电话录音等非音乐文件混入。

## 说明

这版是 ARMusic 的起点版本，还会继续改 bug 和加入视频相关功能。原 LMusic 的保底使用不受影响。
