# 路线图

## 第一版

- 整理项目结构，保留 LMusic 的 Android 基础。
- 新建桌面版播放器骨架。
- 定义局域网同步清单和 `syncId`。
- Android 版增加同步入口页面。
- 桌面版和 Android 版先跑通清单交换。
- 新增歌曲双向同步。

## 第二版

- 桌面版接入真实本地音乐扫描和播放。
- Android 版内置音乐标签编辑页，替换外部“音乐标签”应用跳转。
- 同步歌词、封面、标签变更。
- 同步播放历史和我喜欢歌单。

## 第三版

- 基于听歌时长、播放次数、最近播放和时间段做推荐。
- 冲突处理界面。
- 可选删除同步。
- 局域网自动发现。
- 桌面版打包成 Windows 安装包。
